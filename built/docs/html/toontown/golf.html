golf.GolfGlobals.calcCupListFromHistory(history)[source]

Return a list of booleans, with True meaning he has the cup.

The first item (index 0)is for 10 trophies won. next is 20 trophies won. Last item is for 30 trophies won.

golf.GolfGlobals.calcTrophyListFromHistory(history)[source]

Return a list of booleans, with True meaning he has trophy.

The first item (index 0) is for 4 courses completed. Last item is for 100 course two wins.
