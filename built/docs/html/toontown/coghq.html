coghq.BanquetTableBase module

class coghq.BanquetTableBase.BanquetTableBase[source]

Bases: object

A base class that holds methods and constants shared between the client and AI versions of the banquet table logic. This class manages diner states and other relevant properties.

HUNGRY

The state of a diner who is hungry.

Type:

int

DEAD

The state of a diner who has died.

Type:

int

EATING

The state of a diner who is eating.

Type:

int

ANGRY

The state of a diner who has become angry from not being fed.

Type:

int

HIDDEN

The state of a diner who is hidden from view in the scene.

Type:

int

INACTIVE

The state of a diner who is waiting for the next battle phase.

Type:

int

ANGRY = 3
DEAD = 0
EATING = 2
HIDDEN = 4
HUNGRY = 1
INACTIVE = 5
